# Sistema de Distribuição de Eletivas

Projeto desenvolvido para automatizar a distribuição de alunos em disciplinas eletivas de uma escola.

## Problema

A escola possui cerca de 50 alunos que escolheram 3 eletivas em ordem de prioridade.  
Cada eletiva possui limite de vagas.

O processo manual levava horas.

## Solução

Sistema desenvolvido em C# que:

- importa dados via Excel
- processa as escolhas dos alunos
- distribui automaticamente respeitando prioridades
- gera planilha final

## Tecnologias utilizadas

- C#
- .NET
- Windows Forms
- ClosedXML
- Excel

## Funcionalidades

- Importar planilha de alunos
- Distribuição automática
- Exportar resultado
- Visualização em tabela
